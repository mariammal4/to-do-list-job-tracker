Job Application Tracker (To-Do List)

Project Description
 This is a simple Job Application Tracker web application. It helps users to add, track, and manage their job applications in one place.

Features
 Add job applications easily
 Mark applications as completed or pending
 Delete unwanted tasks
 Simple and user-friendly interface
 
Technologies Used
 HTML
 CSS
 JavaScript
 
How to Use
 Open the project in a browser
 Add your job application details
 Track your progress easily
 
Purpose
 This project is created for practice and to improve web development skills.
